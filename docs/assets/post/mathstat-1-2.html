1.2 Set Theory

Definition 1.2.1. If each element of set \(C_{1}\) is also an element of set \(C_{2}\), then \(C_{1}\) is called \( subset \) of \(C_{2}\), and denoted by \( C_{1} \subset C_{2} \).

Definition 1.2.2. If a set \( C \) has no elements, \( C \) is called the \( null(empty) \) set, and denoted by \( C = \emptyset \).

Definition 1.2.3. The set of all elements that belong to at least one of \( C_{1} \) and \( C_{2} \) is called the \( union \) of \( C_{1} \) and \( C_{2} \), and denoted by \( C_{1} \cup C_{2} \) and it can be generalized to any number of sets. For example, \( C_{1} \cup C_{2} \cup \cdots \cup C_{n} := \cup_{k=1}^{\infty} C_{k} \).

Example 1.2.1. Let \( C_{k} = \left\{ x: \frac{1}{k+1} \le x \le 1\right\} \), then \( \bigcup_{k=1}^{\infty} C_{k}= \left\{ x: 0 < x \le 1 \right\} \).

Definition 1.2.4. The set of all elements that belong to each of \( C_{1} \) and \( C_{2} \) is called the intersection of \( C_{1} \) and \( C_{2} \). and denoted by \( C_{1} \cap C_{2} \), and it can be generalized to any number of sets \( C_{1} \cap C_{2} \cap \cdots \cap C_{n} := \cap_{k=1}^{\infty} C_{k} \).

Example 1.2.2. Let \( C_{k} = \left\{ x: 0 < x < \frac{1}{k} \right\} \), then \( \bigcap_{k=1}^{\infty} C_{k}= \emptyset \).

Definition 1.2.5. Let \( C \) be a subset of \(\mathcal{C}\), then the set that consists of all elements of \(\mathcal{C}\) that are not element of \( C \) is called \(complement\) of \( C \), and denoted by \( C^\complement \) or \( \overline{C} \).

  • A function is called \(point\) or \(set function\) if its domein is point or set, respectively.

Example 1.2.3.
point function: \( f(x) = 2x, f(1)=2 \).
set function: \( Q(A) = \) # of positive integers in \( A \), \( A = \left\{ x: -\infty < x < 6\right\} \Rightarrow Q(A) = 5\).

  • The symbol
    \( \int_{C}^{}f(x)dx \)
    means the ordinary Riemann integral of \(f(x)\) over a one-dimensional set \(C\), the symbol
    \(\int_{}^{}\int_{C}^{} g(x, y)dxdy\)
    means the Riemann integral of \(g(x,y)\) over a two-dimensional set \(C\). Similarly, one or two-dimensional sum is
    \(\sum_{C}^{} f(x), \sum_{}^{}\sum_{C}^{} g(x,y)\).

Example 1.2.4. Let \(Q(C) = \int_{C}^{} \cdots \int_{}^{} dx_{1}dx_{2} \cdots dx_{n}\). If \(C=\left\{ \left( x_{1}, x_{2}, \cdots, x_{n} \right): 0 \le x_{1} \le x_{2} \le \cdots \le x_{n} \le 1 \right\}\), then
\(Q(C) = \int_{0}^{1} \int_{0}^{x^{n}} \cdots \int_{0}^{x^{3}} \int_{0}^{x^{2}} dx_{1} dx_{2} \cdots dx_{n-1} dx_{n} = \frac{1}{n!}\).
