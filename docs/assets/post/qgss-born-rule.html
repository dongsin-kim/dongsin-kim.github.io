[QGSS] 3. Born rule

Qiskit Global Summer School (QGSS) - A key postulate of quantum mechanics which gives the probability that a measurement of a quantum system will yield a given result

Boen Rule

The probability that a state \( \left| \psi \right\rangle \) collapses during a projective measurement onto the basis \( \left\{ \left| x \right\rangle, \left| x^{\perp} \right\rangle\right\} \) to the state \( \left| x \right\rangle \) is given by: \(P\left( x\right)=\left| \left\langle x | \psi \right\rangle \right|^{2}, \sum_{i}^{}P\left( x_{i} \right) = 1. \)

Example 1. \( \left| \psi \right\rangle = \frac{1}{\sqrt{3}}\left( \left| 0 \right\rangle+ \sqrt{2} \left| 1 \right\rangle \right) \) is measured in the basis \( \left\{ \left| 0 \right\rangle, \left| 1 \right\rangle\right\} \). Determine \( P\left( 0\right) \) and \( P\left( 1 \right)\).

\( P\left( 0 \right) = \left| \left\langle 0 \right| \frac{1}{\sqrt{3}}\left( \left| 0 \right\rangle+ \sqrt{2} \left| 1 \right\rangle \right) \right|^{2} = \left| \frac{1}{\sqrt{3}}\left\langle 0 | 0 \right\rangle + \sqrt{\frac{2}{3}} \left\langle 0 | 1 \right\rangle \right|^{2} = \frac{1}{3} \)

\( P\left( 1 \right) = \left| \left\langle 1 \right| \frac{1}{\sqrt{3}}\left( \left| 0 \right\rangle+ \sqrt{2} \left| 1 \right\rangle \right) \right|^{2} = \left| \frac{1}{\sqrt{3}}\left\langle 1 | 0 \right\rangle + \sqrt{\frac{2}{3}} \left\langle 1 | 1 \right\rangle \right|^{2} = \frac{2}{3} \)

Example 2. \( \left| \psi \right\rangle = \frac{1}{\sqrt{2}}\left( \left| 0 \right\rangle - \left| 1 \right\rangle \right) \) is measured in the basis \( \left\{ \left| + \right\rangle, \left| - \right\rangle\right\} \). Determine \( P\left( + \right) \) and \( P\left( - \right)\).

\( P\left( + \right) = \left| \left\langle + | \psi \right\rangle \right|^{2} = \left| \frac{1}{\sqrt{2}}\left( \left\langle 0 \right| + \left\langle 1 \right| \right) \frac{1}{\sqrt{2}}\left( \left| 0 \right\rangle - \left| 1 \right\rangle \right) \right|^{2} = \frac{1}{4} \left| \left\langle 0 | 0 \right\rangle - \left\langle 0 | 1 \right\rangle + \left\langle 1 | 0 \right\rangle - \left\langle 1 | 1 \right\rangle \right|^{2} = 0 \)

\( P\left( - \right) = \left| \left\langle - | \psi \right\rangle \right|^{2} = \left| \frac{1}{\sqrt{2}}\left( \left\langle 0 \right| - \left\langle 1 \right| \right) \frac{1}{\sqrt{2}}\left( \left| 0 \right\rangle - \left| 1 \right\rangle \right) \right|^{2} = \frac{1}{4} \left| \left\langle 0 | 0 \right\rangle - \left\langle 0 | 1 \right\rangle - \left\langle 1 | 0 \right\rangle + \left\langle 1 | 1 \right\rangle \right|^{2} = 1 \)

Please note that:

  • \( \perp \): the perpendicular symbol,
  • \( \left| x^{\perp} \right\rangle \): the state that is orthogonal to \( \left| x \right\rangle \),
  • \( \left\langle 0 | 0 \right\rangle = 1 \cdot 1 + 0 \cdot 0 = 1 \),
  • \( \left\langle 0 | 1 \right\rangle = 1 \cdot 0 + 0 \cdot 1 = 0 \),
  • \( \left\langle 1 | 0 \right\rangle = 0 \cdot 1 + 1 \cdot 0 = 0 \),
  • \( \left\langle 1 | 1 \right\rangle = 0 \cdot 0 + 1 \cdot 1 = 1 \).
