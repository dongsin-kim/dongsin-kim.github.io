[QGSS] 4. Bloch Sphere

Qiskit Global Summer School (QGSS) - A geometrical representation of the normalized and pure quantum state

Bloch Sphere

Let's consider a normalized and pure quantum state as $$ \left| \psi \right\rangle = \alpha \left| 0 \right\rangle + \beta \left| 1 \right\rangle $$ where $$\begin{align} \alpha, \beta &\in \mathbb{C}, \\ \left| \alpha \right|^{2} + \left| \beta \right|^{2} &= 1. \end{align}$$

Since the complex numbers (\(\alpha\) and \(\beta\)) have a real part and an imaginary part, we would need \(4\)-dimensions to represent them all together. To solve this issue, we write the quantum state as $$\begin{aligned} \left| \psi \right\rangle &= \left\{ r_{\alpha} e^{i \varphi_{\alpha}} \left| 0 \right\rangle + r_{\beta} e^{i \varphi_{\beta}} \left| 1 \right\rangle \right\} \times e^{-i \varphi_{\alpha}} \\ &\equiv r_{\alpha} \left| 0 \right\rangle + r_{\beta} e^{i \left( \varphi_{\beta} - \varphi_{\alpha} \right)} \left| 1 \right\rangle \end{aligned}$$ where $$\begin{align} r_{\alpha}, r_{\beta} &\in \left[0, 1\right], \\ r_{\alpha}^{2} + r_{\beta}^{2} &= 1. \end{align}$$

We introduce \(\theta\) using the relationship (Figure 1): $$\begin{align} r_{\alpha} &= \cos \left( \theta / 2 \right), \\ r_{\beta} &= \sin \left( \theta / 2 \right), \\ \theta &\in \left[0, \pi \right]. \end{align}$$

Finally, we can write the quantum state as $$ \left| \psi \right\rangle = \cos \left( \frac{\theta}{2}\right) \left| 0 \right\rangle + e^{i\varphi} \sin \left( \frac{\theta}{2} \right) \left| 1 \right\rangle $$ where $$\begin{align} \varphi_{\beta} - \varphi_{\alpha} &:= \varphi, \\ \varphi &\in \left[ 0, 2\pi\right). \end{align}$$

\(\varphi\) describes the relative phase and \(\theta\) determines the probability to measure \(\left| 0 \right\rangle\) and \(\left| 1 \right\rangle\): $$\begin{align} P\left( 0\right) &=\left| \left\langle 0 | \psi \right\rangle \right|^{2} = \cos^{2} \left( \theta / 2 \right),\\ P\left( 1\right) &=\left| \left\langle 1 | \psi \right\rangle \right|^{2} = \sin^{2} \left( \theta / 2 \right). \end{align}$$

Therefore, all the normalized and pure quantum states can be illustrated on the surface of a sphere with radius \(\left| \vec{r} \right| = 1\), which we call the Bloch sphere (Figure 2) and the coordinates of such a quantum state are given by the Bloch vector: $$ \vec{r} = \left( \begin{array}{c} \sin\theta \cos\varphi \\ \sin\theta \sin\varphi \\ \cos\theta \end{array}\right) $$

For examples,

  • \( \left| 0 \right\rangle \): \(\theta = 0, \varphi = arbitrary, \vec{r} = \left( \begin{array}{ccc} 0 & 0 & 1 \end{array}\right)^{\top} \),
  • \( \left| 1 \right\rangle \): \(\theta = \pi, \varphi = arbitrary, \vec{r} = \left( \begin{array}{ccc} 0 & 0 & -1 \end{array}\right)^{\top} \),
  • \( \left| + \right\rangle \): \(\theta = \left(\pi / 2\right), \varphi = 0, \vec{r} = \left( \begin{array}{ccc} 1 & 0 & 0 \end{array}\right)^{\top} \),
  • \( \left| - \right\rangle \): \(\theta = \left(\pi / 2\right), \varphi = \pi, \vec{r} = \left( \begin{array}{ccc} -1 & 0 & 0 \end{array}\right)^{\top} \),
  • \( \left| +i \right\rangle \): \(\theta = \left(\pi / 2\right), \varphi = \left(\pi / 2\right), \vec{r} = \left( \begin{array}{ccc} 0 & 1 & 0 \end{array}\right)^{\top} \),
  • \( \left| -i \right\rangle \): \(\theta = \left(\pi / 2\right), \varphi = \left(3\pi / 2\right), \vec{r} = \left( \begin{array}{ccc} 0 & 0 & 1 \end{array}\right)^{\top} \).
