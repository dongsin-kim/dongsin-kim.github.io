How to type science equations in LibreOffice?

A quick guide to install TexMaths extension

Install TexMaths

1. Open [LibreOffice Writer]

2. Find [Tools] --> [Extension Manager]

3. Click [Get more extensions online...]

4. Search [TexMaths]

5. Click & Download a latest release

6. Move to [Downloads] directory

7. Double-click [texmaths-0-*.oxt]

8. Click [OK] and [Restart Now]

Configure TexMaths

1. Open a terminal

2. Type as follows:

1
sudo apt-get install texlive texlive-extra-utils
cs

3. Press the [Enter] button

4. Open [LibreOffice Writer]

5. Click [TexMaths System Configuration]

6. Move to [Shortcuts]

7. Set [Equations Shortcuts]

Note:

Use the .odt format to save your files. Microsoft office does not support '.svg' images.

Do not use '.png' as the print quality will never be as good as '.svg'.
