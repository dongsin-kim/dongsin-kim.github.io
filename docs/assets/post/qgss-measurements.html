Qiskit Global Summer School (QGSS) - Once we measure the superposition state, it collapses to one of its states

Measurements

We choose orthogonal bases to describe and measure quantum states. This is called projective measurements.

During a measurement onto the bases \( \left\{ \left| 0 \right\rangle , \left| 1 \right\rangle \right\} \), the state will collapse into either state \( \left| 0 \right\rangle \) or \( \left| 1 \right\rangle \). As these are the eigenstates of \(\sigma_{z}\), we call this a \(z\)-measurement.

There are infinitely many different bases, but other common ones are \( \left\{ \left| + \right\rangle, \left| - \right\rangle\right\} \) and \( \left\{ \left| +i \right\rangle, \left| -i \right\rangle\right\} \), corresponding to the eigenstates of \(\sigma_{x}\) and \(\sigma_{y}\), respectively.

Please note that:

  • \( \left| 0 \right\rangle := \left( \begin{array}{c} 1 \\ 0 \end{array} \right) \)
  • \( \left| 1 \right\rangle := \left( \begin{array}{c} 0 \\ 1 \end{array} \right) \)
  • \( \left| + \right\rangle := \frac{1}{\sqrt{2}} \left( \left| 0 \right\rangle + \left| 1 \right\rangle \right) \)
  • \( \left| - \right\rangle := \frac{1}{\sqrt{2}} \left( \left| 0 \right\rangle - \left| 1 \right\rangle \right) \)
  • \( \left| +i \right\rangle := \frac{1}{\sqrt{2}} \left( \left| 0 \right\rangle + i \left| 1 \right\rangle \right) \)
  • \( \left| -i \right\rangle := \frac{1}{\sqrt{2}} \left( \left| 0 \right\rangle - i \left| 1 \right\rangle \right) \)
